---------------------------------------------------------
			TrMnL
---------------------------------------------------------
					- Rohit Talwar

This is emulates a terminal of a linux system, with some
added functionalities -
1. "hist" - list out all the commands listed before
2. The directory from which the a.out is executed that 
   directory is considered as the home directory
3. "!hist n" lists the last n commands
4. Custom commands can be added to existing code
